AttenDuino(attendance recording device)

This project is an electronic attendance recording device developed as part of our Practical Research 1 course. The system was designed to automate the process of recording student attendance, reduce manual errors, and improve efficiency in classroom monitoring. It utilizes sensors and basic electronic components to detect user input and store attendance data systematically. The goal of the project was to create a simple yet functional prototype that demonstrates how technology can support academic record-keeping. - Developed collaboratively as part of a Practical Research 1 group project

Technologies Used:Microcontroller (e.g., Arduino or similar) RFID/NFC or Input Sensors Basic Electronic Components Breadboard and Wiring LCD Display Arduino IDE or similar software
